APIs For Food Order App
Authentication
  • User Sign Up : domain/authApis/sign-up
  • User Sign In : domain/authApis/sign-in
Register A Cafe
  • Register : domain/cafeApis/createCafe
  • Form Input : name, address
  • Token Input : cafe [user_id]
Item Category
  • Category List : domain/itemApis/category-list
  • Category Detail : domain/itemApis/category-detail/id
  • Category Create : domain/itemApis/category-create
  • Form Input : name, address
  • Token Input : user [user_id], cafe [cafe_id]
  • Category Update : domain/itemApis/category-update/id
  • Category Delete : domain/itemApis/category-delete/id
Items
  • Item List : domain/itemApis/item-list
  • Item Detail : domain/itemApis/item-detail/id
  • Item Create : domain/itemApis/item-create
  • Item Update : domain/itemApis/item-update/id
  • Item Delete : domain/itemApis/item-delete/id
